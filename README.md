# OBS-Agent

基于多模态 AI 的桌面操控智能体。通过 OBS 截取屏幕 → 多模态 AI 分析 → 模拟鼠标键盘操作，实现自动化任务执行。

---

## 功能

- 🖥️ **屏幕感知**：通过 OBS WebSocket 实时截图，叠加像素刻度网格辅助 AI 精确定位
- 🧠 **AI 决策**：调用多模态大模型分析截图并返回结构化动作指令
- 🖱️ **键鼠操控**：pyautogui 执行点击、移动、输入、快捷键等操作
- ⏰ **定时任务**：支持每天/每小时自动执行，任务持久化存储
- 🖼️ **GUI 界面**：Tkinter 图形界面，实时日志、配置管理、任务列表
- 🛡️ **安全机制**：步数熔断、FailSafe 紧急停止、手动中断

## 快速开始

### 1. 环境要求

- Python 3.10+
- OBS Studio 28+（需开启 WebSocket 服务）
- Windows 10/11

### 2. 安装依赖

```bash
pip install openai obsws-python pyautogui pyperclip pillow
```

### 3. 配置

首次运行会自动引导填写配置，或在gui里进行编辑：

```ini
[key]
api_key = 你的API密钥
base_url = 你的api提供方
model = 你的模型

[obs]
host = 127.0.0.1
port = 4455
password = 你的OBS密码
scene = 场景
```

### 4. 启动 OBS

- 打开 OBS Studio
- 菜单：工具 → WebSocket 服务器设置 → 启用
- 设置密码，默认端口 4455
- 确保有一个包含「显示捕获」源的场景

### 5. 运行


双击 `go obs-agent.bat` 一键启动。

### 定时任务

支持两种模式：

- **每天**：在指定时刻执行（如 `09:00`）
- **每小时**：在指定分钟执行（如 `30` 表示每小时的 30 分）

任务保存在 `tasks.json`，重启不丢失。

## 工作原理

```
┌──────────┐     ┌──────────┐     ┌──────────┐
│ OBS 截图  │ ──▶ │  AI 分析  │ ──▶ │ 执行动作  │
│ +网格刻度 │     │ 返回JSON │     │ 鼠标/键盘 │
└──────────┘     └──────────┘     └──────────┘
       ▲                                │
       └──────── 循环 (最多 N 步) ───────┘
```

每步流程：

1. OBS 截图 → 叠加 100px/50px 红色坐标刻度 → 转为 base64
2. 发送给多模态 AI（含系统提示词 + 操作历史）
3. AI 返回 JSON：`{"thinking":"...","action":{"type":"click","x":320,"y":1050}}`
4. 解析并执行动作（move / click / type / key 等）
5. 记录历史，进入下一步

## 故障排除

| 问题 | 解决 |
|------|------|
| OBS 连接失败 | 检查 OBS → 工具 → WebSocket 服务器 → 已启用，密码端口正确 |
| 截图为空 | 确保场景包含「显示捕获」源且有画面 |
| AI 返回格式错误 | 检查 API Key 和模型名，确认模型支持多模态 |
| 鼠标位置偏差大 | 更换更高性能的多模态模型 |
| FailSafe 触发 | 鼠标移到屏幕四角是紧急停止，正常现象 |
